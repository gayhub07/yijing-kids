## 🔍 数据库验证

在 Supabase Dashboard → SQL Editor 执行：

```sql
-- 查看签到记录
SELECT * FROM daily_signins ORDER BY created_at DESC LIMIT 10;

-- 查看表结构
\d daily_signins

-- 检查 RLS 策略
SELECT * FROM pg_policies WHERE tablename = 'daily_signins';
```

## 🐛 可能的问题

### 问题 1：签到失败
**原因**：数据库表未创建成功
**解决**：重新执行 migration SQL

### 问题 2：类型错误
**原因**：TypeScript 类型定义未更新
**解决**：
```bash
pnpm check
```

### 问题 3：样式问题
**原因**：浏览器缓存
**解决**：硬刷新 (Ctrl+Shift+R)
